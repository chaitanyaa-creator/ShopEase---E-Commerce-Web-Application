## вљ™пёЏ How It Works

### Product Management

* Product information is fetched dynamically from the Fake Store API using the Fetch API.
* Products are rendered dynamically using JavaScript DOM manipulation.

### Search Functionality

* Users can search products by entering keywords in the search bar.
* Product results are filtered instantly without reloading the page.

### Category Filtering

* Users can filter products by category such as:

  * Men's Clothing
  * Women's Clothing
  * Electronics
  * Jewelry

### Shopping Cart

* Users can add products to their cart.
* Cart count updates dynamically.
* Cart data is stored in Local Storage, ensuring persistence after page refreshes.

### Cart Page

* Displays all selected products.
* Allows users to remove products from the cart.
* Calculates and displays the total cart value.
